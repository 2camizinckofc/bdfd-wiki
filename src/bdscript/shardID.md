# $shardID

Returns the id of the currently used shard.


## Usages

There are two usages of  the ` $shardID ` function.

### Usage #1

` $shardID ` - Returns the shard id of the current guild.

### Usage #2

` $shardID[guildID] ` - Returns the shard id of the provided guild.

## Example

```
$nomention
$shardID
```
