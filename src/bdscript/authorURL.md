# $authorURL
Adds a hyperlink to the author text.

## Usage
```
$authorURL[link]
```
> $authorURL cannot work without $author.

## Example
```
$nomention
$author[Click me to visit the BDFD website!]
$authorURL[https://botdesignerdiscord.com]
```
