# $minute
Returns the current minute.
> Tip: You can use [$time](https://nilpointer-software.github.io/bdfd-wiki/bdscript/time.html) to change the timezone.

## Usage
```
$minute
```

## Example
```
$nomention
Current Minute: $minute
```
