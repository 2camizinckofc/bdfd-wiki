# $isBoolean
Returns whether or not the provided 'text' is a boolean.
> 'true' means the text is a boolean, 'false' means it isn't.

## Supported Booleans
Positive |  Negative 
-------- | --------
true | false
yes | no
on | off
enable | disable

## Usage
```
$isBoolean[text]
```

## Breakdown
`text` - The text to check.
