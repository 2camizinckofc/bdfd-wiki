# $replyIn
The bot waits x *(amount of time)* before executing the code.

## Usage
```
$replyIn[delay]
```
> 🧙‍♂️ The 'delay' can't be more than 40 minutes or less than 1 second.

### Breakdown
- `delay` - How long the command is delayed (e.g: 10s, 10m, 40m).

## Example
```
$replyIn[5s]
Hi $username!
```
