# $replyIn
The bot waits [x] amount of time before executing the command.

## Usage
```$replyIn[delay]```

### Breakdown
`delay` - How long the command is delayed (e.g: 10s, 10m, 40m)

## Example
```
$replyIn[5s]
Hi $username!
```
