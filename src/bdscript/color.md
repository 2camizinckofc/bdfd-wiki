# $color
Sets the embed border color.

## Usage
```
$color[colorHex]
```

### Breakdown
`colorHex` - The [color hex](https://htmlcolorcodes.com/color-picker) to set the embed border color as.

## Example
```
$nomention
$description[⬅️ That is the embed color border!]
$color[#673ab7]
```
