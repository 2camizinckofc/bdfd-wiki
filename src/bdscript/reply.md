# $reply
The bot replies to the user's command. This feature makes use of Discord's quoting/replying feature, learn more [here](https://support.discord.com/hc/en-us/articles/360057382374-Replies-FAQ).

## Usage
```$reply```

## Example
```
$nomention
$reply
Hello $username!
```
