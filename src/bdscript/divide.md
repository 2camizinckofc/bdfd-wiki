# $divide
Divides the provided numbers.

## Usage
```
$divide[numbers]
```

### Breakdown
`numbers` - The numbers to divide. Separate numbers using `;`.

## Example
```
$nomention
$argsCheck[>2;❌ Please provide the needed arguments! Usage: `!divide (number1) (number2)`]
Answer: $divide[$message[1];$message[2]]
```
